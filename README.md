# Shimul's Portfolio

A personal portfolio website showcasing academic achievements, skills, projects, and contact information.

## Features
- Clean, responsive design
- Smooth scroll animations
- Fixed navigation bar
- Mobile-friendly layout

## Tech Stack
- React
- CSS
- HTML

## Getting Started

```bash
# Clone the repo
git clone https://github.com/ShimulCoding/Shimuls-Portfolio.git

# Navigate to folder
cd Shimuls-Portfolio

# Install dependencies
npm install

# Run locally
npm start
```

## Deployment

The site is deployed via GitHub Pages and auto-updates on every push to `main`.

🔗 Live URL: https://shimulcoding.github.io/Shimuls-Portfolio/
